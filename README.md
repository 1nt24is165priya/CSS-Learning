CSS Chapter 2 Practice
A simple HTML + CSS project built to practice selectors, pseudo-classes, and styling techniques. The layout is inspired by a Quora-style feed with posts, navigation, and interactive elements.

Features:
Multiple CSS files (style2.css, style3.css) to explore specificity & overrides

Use of element, class, ID, and attribute selectors

Practice with combinators (+, >, nth-of-type)

Interactive styles using :hover, :active, :checked

Creative effects with ::first-line, ::first-letter, ::selection
